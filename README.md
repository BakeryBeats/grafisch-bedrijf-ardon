# Grafisch Bedrijf Ardon - Website

Moderne, responsive website voor Grafisch Bedrijf Ardon.

## Functies

- ✨ Modern en responsief ontwerp
- 📱 Mobiel-friendly layout
- 🎨 Aantrekkelijke animaties
- ⚡ Snel laden
- ♿ Toegankelijk design

## Secties

1. **Home** - Heroafbeelding met call-to-action
2. **Over ons** - Bedrijfsinfo met features
3. **Diensten** - Overzicht van aangeboden diensten
4. **Portfolio** - Galerij met projecten
5. **Contact** - Contactformulier
6. **Footer** - Copyright info

## Technologie

- HTML5
- CSS3 (Flexbox & Grid)
- Vanilla JavaScript
- Google Fonts (Poppins)

## Lokaal uitvoeren

1. Clone de repository
2. Open `index.html` in een web browser
